# Developer Portfolio

Personal portfolio site. Live at [anishpatel.xyz](https://anishpatel.xyz)

## Tech Stack:

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Background:** Vanta.js (Three.js)
- **Icons:** React Icons
- **Analytics:** Vercel Analytics
- **Deployment:** Vercel
